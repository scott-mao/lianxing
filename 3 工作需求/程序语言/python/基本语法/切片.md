





在python中对于数组或者张量可以通过切片操作取出数组中相应维度的数据

- 其中``:``代表该维度的所有数据
- ``...``用来取前面所有维度数据

例如：

```python
a = torch.tensor([[1,2],[3,4],[0,1]])
a[:,1] 
>> [1,3,0]
a[:,0:1]  #使用这种方法读取可以保持原来维度不变
>> [[1],[3],[0]]
a[...,1]
>> [1,3,0]
```



注意：列表中的切片不能取出指定列的数据，可以通过列表解析的方式读取，详见[python中数组和列表读取一列的方法](http://www.kongxiangbo.com/?p=692)
