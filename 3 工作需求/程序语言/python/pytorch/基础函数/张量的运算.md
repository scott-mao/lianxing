# 张量的拼接与拆分

## ``torch.cat``

该函数在在指定``dim``对函数进行拼接，拼接之后不会增加新的维度，是原有维度相加

```python
x = torch.randn(2,3)
y = torch.randn(3,3)
z = torch.cat((x,y),0)
print(z.size())
z为(5,3)
```

## ``torch.stack``

创建新的维度并在该指定维度上进行拼接

```
x = torch.randn(2,3)
y = torch.randn(2,3)
z = torch.stack((x,y),0)
print(z.size())
print(z)
此z为(2,2,3)
```
